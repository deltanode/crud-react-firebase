# CRUD App

A basic app to perform CRUD operation using React & Firebase

### Topics Coverd:-

While creating the app covered:
- From Firebase:
  - `addDoc()` & `collection()`
  - `query()` & `onSnapshot()`
  - `updateDoc()` & `doc()`
  - `deleteDoc()`
- using `env-cmd` for `.env` & `.env.dev`
- using `react-icons`
